To create form configurations for multiple languages, this tool needs both the Excel translation file and the form configuration files for different languages. The form configurations must be named after the language key to be used for the translation (e.g. 'de' or 'en').

If only one form configuration file is given, it is considered to be the blueprint for all translation keys. It will then create form configuration files for all languages that are present in the Excel file based on the one form configuration file given.

Could not generate form configurations.

Please use names for the form configuration files that reflect the language each file represents e.g. en.json or de.json. If you do not follow this convention, it will be harder for the translators to associate their translations with the right language.
